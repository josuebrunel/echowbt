EchoWBT
=======

**EchoWBT** is a simple wrapper of *httptest* which helps test *Echo* apps easily

Installation
------------

.. code:: go
    
    go get github.com/josuebrunel/echowbt


Quickstart
----------

.. code:: go

    package pkg_test

    import (
        "github.com/josuebrunel/echowbt"
        "testing"
    )
